# Public Project Notes

`Project_Docs/` is reserved for sanitized public notes about quality, experiments, and research-facing documentation.

Internal run ledgers, deep review packs, and operational tracking material are not maintained in the public repository.

Public provenance and boundary notes live here when they are safe to publish.
Private companion modules, customer-specific policies, release authority state,
and internal pricing or evaluator collateral do not belong in this tree.
